**1. 前缀、中缀、后缀**
```c++
/*
	+ - * / % > < == >= <= && || ! 等
*/
1 + 2; // 3 中缀
+ 1 2; // 3 前缀
1 2 +; // 3 后缀

1 + 2 - 4; // -1 中缀
- + 1 2 4; // -1 前缀（先执行+ 1 2，结果为3，再执行- 3 4，结果为-1）
1 2 + 4 -; // -1 后缀（先执行+ 1 2，结果为3，再执行3 4 -，结果为-1）

1 * (4 - 2); // 2 中缀
* 1 () - 4 2; // 2 前缀
```

**2. 二叉树**
```
       1
   2       3
4     5 6     7

先序：1 2 4 5 3 6 7
中序：4 2 5 1 6 3 7
后序：4 5 2 6 7 3 1
```
